# Auditoria Interna de TI — Botium Toys (Simulação)

**Resumo**  
Projeto de auditoria interna realizado como exercício prático (Google Cybersecurity Certificate). Avaliação baseada no **NIST Cybersecurity Framework** com foco em ativos, controles técnicos/administrativos/físicos e conformidade (PCI DSS, GDPR, SOC).

**Entregáveis**
- `relatorio-final.pdf` — Relatório técnico completo da auditoria.  
- `escopo-e-objetivos.md` — Escopo e metas da auditoria.  
- `avaliacao-de-riscos.md` — Avaliação de riscos (inclui pontuação).  
- `lista-de-controles.md` — Lista de verificação de controles preenchida.  
- `conformidade.md` — Verificação de conformidade (PCI, GDPR, SOC).  
- `recomendacoes.md` — Recomendações e plano de ação.

**Principais achados**
- Ausência de criptografia para dados de cartão de crédito.  
- Falta de IDS e de backups regulares.  
- Políticas de senha fracas; sem gerenciador centralizado.  
- Controles físicos (fechaduras, CFTV, alarme de incêndio) OK.

**Como usar**
1. Navegue pelos arquivos `.md` para a documentação.  
2. Consulte `relatorio-final.pdf` para versão formatada pronta para anexar a currículo.  
3. Use o repositório como portfólio e referência técnica em entrevistas.
